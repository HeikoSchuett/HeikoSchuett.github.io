Code & Data

Code and data for most of my projects can be found online. If you miss anything, please write me an E-Mail.


rsatoolbox

rsatoolbox is a toolbox for representational similarity analysis
The code for the toolbox is available at github .

Our paper on the new statistical inference methods is published in Elife here.


Psignifit 4

Psignifit is a toolbox for fitting Psychometric functions.
The code for the toolbox is available at github .
Help and explanations are available in the corresponding wiki .

Our paper explaining the methodological advancements is published in vision research and freely available here.


Early spatial vision model

This model simulates the earliest steps of vision processing on arbitrary luminance images.
The code for the model is available at github .
A basic intro to the code is available in the corresponding wiki .

Our paper explaining the model is published in journal of vision and freely available here
